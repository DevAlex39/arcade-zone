# 🎲 Yahtzee

Yahtzee classique 1 à 4 joueurs en HTML/CSS/JS pur.

## Fonctionnalités

- 1 à 4 joueurs avec noms personnalisables
- Dés au design classique (points positionnés à l'identique des vrais dés)
- Animations de lancer 3D (3 variantes aléatoires)
- Prévisualisation des combinaisons disponibles en temps réel
- Tableau de scores complet avec toutes les catégories Yahtzee
- Écran de fin avec podium

## Structure

```
index.html   — interface (setup joueurs + plateau de jeu)
style.css    — thème sombre, animations dés
game.js      — logique complète du Yahtzee
```

## Lancer

Ouvrez `index.html` dans un navigateur. Aucune dépendance.
